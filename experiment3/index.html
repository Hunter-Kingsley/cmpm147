Experiment 03: Alternate Worlds

Description

Generates an original style rogue dungeon with auto-tiling features, create another generative-tiled overworld scene.

Technical

For every tile(symbol in ascii table) that needs to be detailed, the code checks the surrounding tiles and creates a bitmask that is passed to a table to pick the correct tile to layer over it

Reflection

I think what I was able to produce from this assignment was definitely pretty neat, but I couldn't help but get stuck on the small details that were left unchecked due to the limitations in place. In the dungeon, there are some obvious tiles in some generation that would have benefited greatly from taking in the context of the 8 tiles around it instead of just 4. In the overworld, I maybe could have had to code do separate passes for each direction to create multiple layers instead of just being stuck with choosing one, but I thought that was a little out of scope.
